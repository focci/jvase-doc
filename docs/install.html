安装 (Install)

常见的几种引入方式

NPM

推荐使用NPM的安装方式，它能更好地配合 Gulp 进行使用.

npm install --save jvase

CDN

可以通过 https://unpkg.com/jvase/dist/ 来浏览最新版本的资源文件，通过引入CSS文件来添加Jvase.

<!-- 最新版本的 Jvase 核心 CSS 文件 -->
<link rel="stylesheet" href="https://unpkg.com/jvase/dist/jvase.min.css" />

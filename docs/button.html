按钮 (Button)

创建多风格的按钮

用法介绍

添加类 .v-btn 至行内元素(比如: a、button).

Link

风格

默认支持7种不同的风格，添加相应的类即可

外边框

添加类 .v-outlined 至 .v-btn，不能与类 .v-white 同时使用.

胶囊按钮

添加类 .v-pill 至 .v-btn

尺寸

除了默认按钮外，还支持3种不同尺寸：.v-small、.v-medium、.v-large

Loading

添加类 .v-loading 至 .v-btn 即可.

图标按钮

Full Width

添加类 .v-fullwidth 至 .v-btn 即可.

按钮组

将多个按钮 .v-btn 添加至类名为 .v-btns容器即可.

胶囊按钮组

添加类 .v-pill 至 .v-btns容器即可.

垂直按钮组

添加类 .v-vertical 至 .v-btns容器即可， 不支持与 .v-pill 同时使用.

如果要设置尺寸，可以添加相应的尺寸类(比如：.v-large)至 .v-btns即可.

按钮组(Full Width)

添加类 .v-vertical、.v-fullwidth 至 .v-btns容器即可.
